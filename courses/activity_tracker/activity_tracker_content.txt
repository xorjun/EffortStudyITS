Print a Welcome Message

We start by learning how programs can display messages on the screen.
Programs often show welcome messages, instructions, or information to the user.

In Python, we use the print() command to display text.

Text should be written inside quotes (" "), so the program knows which part is text.

Example
print("How are you")

This will output:

How are you

Writing a Function

A function is a small block of code that performs one task.

In Python, we create a function using the def keyword.

Example
def say_hello():
    print("Hello")

This creates a function named say_hello.

The code inside the function will only run when the function is called.

Calling the Function:
say_hello()

This will output:

Hello

2.Remember a Study Subject

Programs often need to remember information so it can be used later.

In Python, we can store information inside a variable.

A variable is a named place where the program keeps something temporarily.

For example, a program can remember:

a study subject
a username
a score
a message
Using Variables

A variable is created using the = symbol.

study_subject = "Python"

This stores the text "Python" inside the variable study_subject.

Using a Function Parameter

Functions can receive information from outside.

The value inside the brackets is called a parameter.

def show_subject(subject):
    print(subject)

Here, the function receives a subject and uses it inside the function.

Returning a Value

A function can give a value back using return.

def get_subject():
    subject = "Python"
    return subject

The function returns the value stored in subject.

Example
def remember_subject(subject):
    study_subject = subject
    return study_subject

If the function is called like this:

remember_subject("Python")

the returned value will be:

Python

3. Start a Session List

Programs often need to store multiple pieces of information together.

In Python, we can use a list to keep many items in one place.

A list can store:

study subjects
session times
names
messages

Lists keep items in order, and new items can be added later.

Creating an Empty List

An empty list is created using square brackets:

activities = []

Right now, the list has no items inside it.

Adding Items to a List

We can add new items using append().

activities = []
activities.append("Reading")

Now the list contains:

["Reading"]
Lists Can Store Multiple Items
activities = []
activities.append("Reading")
activities.append("Exercise")
activities.append("Music")

This creates:

["Reading", "Exercise", "Music"]
Using Lists Inside Functions

A function can create a list and return it.

def create_activities():
    activities = []
    return activities

When the function runs, it returns an empty list.

4. Add One Study Session

Sometimes one piece of information is not enough.

A study session may need to store:

the subject
the study duration

In Python, a list can store multiple values together.

A list can even contain another list inside it.

Storing Multiple Values Together
session = ["Mathematics", 45]

This list stores:

"Mathematics" → the subject
45 → the duration
Lists Inside Lists

A larger list can contain many study sessions.

sessions = []

sessions.append(["Mathematics", 45])
sessions.append(["Physics", 30])

Now the list becomes:

[["Mathematics", 45], ["Physics", 30]]

Each inner list represents one study session.

Converting Text to a Number

Sometimes numbers are stored as text.

Example:

"45"

This is text, not a real number.

To convert text into a number, we use int().

duration = int("45")

Now duration becomes the number:

45
Using This in a Function

A function can:

convert the duration
create a study session
add it to the list
return the updated list
Example
def add_movie(movies, name, year_text):
    year = int(year_text)

    movies.append([name, year])

    return movies

Calling the function:

add_movie([], "Inception", "2010")

returns:

[["Inception", 2010]]

5. Show All Study Sessions

Programs often need to repeat the same action multiple times.

For example, a study tracker may need to:

show every study session
display every subject
process all stored data

In Python, we use a for loop to go through items one by one.

Using a for Loop

A for loop takes one item from a list at a time.

colors = ["Red", "Blue", "Green"]

for color in colors:
    print(color)

This will output:

Red
Blue
Green
Looping Through Study Sessions

A study session list can look like this:

sessions = [
    ["Mathematics", 45],
    ["Physics", 30]

Each item contains:

a subject
a duration
Accessing Values Inside Each Session

Inside the loop:

session[0] gives the subject
session[1] gives the duration

Example:

session = ["Mathematics", 45]

print(session[0])
print(session[1])

This outputs:

Mathematics
45
Printing Formatted Messages

We can combine text and values inside print().

subject = "Physics"
duration = 30

print("Subject:", subject, "- Duration:", duration)

This outputs:

Subject: Physics - Duration: 30
Example
def show_fruits(fruits):
    for fruit in fruits:
        print("Fruit:", fruit)

Calling:

show_fruits(["Apple", "Banana"])

will print:

colors = ["Red", "Blue", "Green"]

Red

the subject

name = "Alice"

message = "Name: " + name + ", Age: " + str(age)

Name: Alice, Age: 20
Converting Numbers to Text

duration = 45

7. Calculate Total Study Time

Programs often need to calculate totals.

For example, a study tracker may want to calculate the total number of minutes studied.

To do this, we use an accumulator variable.

An accumulator starts with a value and is updated inside a loop.

Starting a Total

Usually, the total starts at 0.

total = 0
Adding Values Inside a Loop
total = total + 10

This updates the total by adding a new value.

Example
numbers = [5, 10, 15]

total = 0

for number in numbers:
    total = total + number

print(total)

This outputs:

30
Using This with Study Sessions

A study session looks like this:

["Mathematics", 45]

Here:

session[0] is the subject
session[1] is the duration

We only add the durations.

Another Example
def total_scores(scores):

    total = 0

    for score in scores:
        total = total + score

    return total

Calling:

total_scores([20, 15, 5])

returns:

40

8. Give a Simple Recommendation

Programs can make decisions based on conditions.

For example, a study tracker can:

* encourage the user to study more
* congratulate the user for studying enough

In Python, we use `if` and `else` for this.



Using `if`:

An `if` statement checks whether something is true.

python id="r4m8qx"
age = 15

if age < 18:
    print("Minor")


This prints:

text id="z7p2la"
Minor


because `15` is less than `18`.

Using `else`:

`else` is used when the condition is not true.

python id="k5v3td"
score = 80

if score < 50:
    print("Try again")
else:
    print("Well done")

This outputs:

text id="u2x9we"
Well done

Returning Messages from Functions:

A function can return different messages depending on a condition.

python id="m8q1rv"
def weather_message(temperature):

    if temperature < 20:
        return "Cold day"
    else:
        return "Warm day"


Calling:

python id="t6p4av"
weather_message(25)


returns:

text id="w3n7lb"
Warm day
