# Welcome to the Activity Tracker Python Course

We are excited to have you explore programming with us. This platform is designed to help beginners learn Python step by step through small learning sections and practical coding tasks.

Our system is currently in an early development stage, so some features and improvements are still being added. While using the platform, you may notice ongoing changes as we continue improving the learning experience.

---

## Introduction to Python

This course is a beginner-friendly introduction to the Python programming language. The goal of this course is not to teach every detail of Python, but to help you understand the basic ideas needed to solve simple programming tasks.

The course focuses on:

- Writing and understanding simple Python programs
- Learning by reading examples
- Solving small coding tasks step by step
- Building parts of a simple Activity Tracker application

You do not need any previous programming experience to complete this course.

---

## How This Course Works

The course is divided into small sections. In each section, you will:

1. Read short learning materials
2. See simple code examples
3. Complete a programming task

The tasks gradually build parts of a small **Activity Tracker** program that can:

- Store activities
- Track time
- Show summaries
- Give simple feedback

The course focuses on practical understanding rather than advanced theory. Explanations are kept simple and use everyday language whenever possible.

---

## Before You Start

While working on the tasks:

- Read the learning content carefully
- Try to understand each example
- Take your time when solving tasks
- Small mistakes are normal while learning programming

Programming is learned through **practice and experimentation**. The goal is not perfection, but understanding how programs work step by step.

We hope you enjoy learning Python with the Activity Tracker course!
