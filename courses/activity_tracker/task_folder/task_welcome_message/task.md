# Print a Welcome Message

We start by learning how programs can display messages on the screen.
Programs often show welcome messages, instructions, or information to the user.

In Python, we use the `print()` command to display text.

Text should be written inside quotes (`" "`), so the program knows which part is text.

## Example

```python
print("How are you")
```

This will output:

```
How are you
```

## Writing a Function

A function is a small block of code that performs one task.

In Python, we create a function using the `def` keyword.

```python
def say_hello():
    print("Hello")
```

This creates a function named `say_hello`.

The code inside the function will only run when the function is **called**.

**Calling the Function:**

```python
say_hello()
```

This will output:

```
Hello
```
