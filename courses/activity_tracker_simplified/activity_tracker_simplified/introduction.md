# Welcome to the Activity Tracker Python Course

This course is designed for complete beginners who have never programmed before.

You do not need any previous coding experience.

In this course, you will slowly build small parts of a simple **Activity Tracker** program step by step.

You will learn:

- how to display messages
- how to organize code into functions
- how to store information in variables
- how to work with multiple activity sessions

---

## How to Use the Platform

**The Coding Area** — You can type your code in the coding area on the right.

**Run Button** — The Run button tests your code.

**Submit Button** — The Submit button sends your final solution.

**AI Hint Button** — If you get stuck, you can use the AI Hint button for guidance.

---

## How This Course Works

The exercises use very small coding steps.

Some code is already written for you.

You only need to complete the missing parts.

Each task introduces one new concept before asking you to apply it.
