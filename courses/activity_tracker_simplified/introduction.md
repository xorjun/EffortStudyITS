# Course Introduction

Welcome to the Activity Tracker Python Course. This course is designed for you to try out programming. It is designed for complete beginners who have never programmed before. You do not need any previous coding experience.

In this course, you will slowly build toward a working program. The program is a simple Activity Tracker, in which you can log what you are doing. You will build toward this step by step.

You will learn:

- How to add comments to program code
- How to display messages
- How to organize code
- How to store information
- How to work with different types of information
- How to work with multiple activity sessions

The exercises use very small coding steps.
Some code is already written for you.
You only need to complete the missing parts.

---

## How to Use the Platform

**The Coding Area** — You can type code in the coding area (right side of the screen).

**Start Program Button** — The Start Program button tests your code (bottom left on the right of your screen).

**Test & Submit Button** — The Test & Submit button sends your final solution (bottom right on the right of your screen).

**AI Help Button** — The AI Help button provides coding suggestions.

---

Each task introduces one new concept before asking you to apply it.
