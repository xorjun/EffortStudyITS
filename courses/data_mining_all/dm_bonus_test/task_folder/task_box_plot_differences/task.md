# Box Plot for Differences

**Note:** This task is related to Bonus Task 01.16 from the first exercise sheet.

Assume you have four arrays available:

1. `pre_test_control`
2. `pre_test_intervention`
3. `post_test_control`, and
4. `post_test_intervention`

containing the data recorded during a study, namely 1) the pre-test results in the control group, 2) the pre-test results in the intervention group, 3) the post-test results in the control group, and 4) the post-test results in the intervention group.

## TODO

In a shared axis, create two box plots using the function `matplotlib.pyplot.boxplot`. The first box plot should be the difference between post- and pre-test scores for the control condition, the second box plot should be the difference between post- and pre-test scores for the intervention condition.
