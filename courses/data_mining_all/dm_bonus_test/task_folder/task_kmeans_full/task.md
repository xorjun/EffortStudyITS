# Full K-means

**Note:** This task is related to Bonus Task 02.16 from the second exercise sheet.

Implement the full K-Means algorithm and apply it to the data in the array `X` for `K = 2`.

Afterwards, provide a scatter plot to show your cluster assignment. The array `Y` contains a 2D-version of the data set (reduced via PCA) that you can use for your plot. The color of each data point should indicate which cluster the data point belongs to.

When you submit your result, the solution plot will show you the result of the `sklearn` implementation of KMeans. Is your result consistent with the `sklearn` result? What are differences?
