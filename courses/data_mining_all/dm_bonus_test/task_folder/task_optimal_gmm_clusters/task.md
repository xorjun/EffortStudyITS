# Guassian Mixture Models

Write a function that receives as input a data matrix `X` and a maximum number of clusters and returns the optimal number of clusters `K` according to the Bayesian information criterion for Gaussian Mixture Models.

**Hint:** You are allowed to use the `scipy` implementation of Gaussian Mixture Models here. Look up the [documentation](https://scikit-learn.org/stable/modules/generated/sklearn.mixture.GaussianMixture.html) to find out how to compute the Bayesian information criterion.
