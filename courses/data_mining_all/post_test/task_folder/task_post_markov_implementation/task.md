# Markov Models: Implementing Bayesian Knowledge Tracing

Consider a Bayesian Knowledge Tracing Model with parameters `p_slip`, `p_guess`, `p_trans`, and `p_start` and let `q` be the  conditional probability of the student having mastered the skill at time t given all successes and failures up to time t.

Then, which code correctly computes the conditional probability of the student having mastered the skill at time t+1 given all successes and failures up to time t?
